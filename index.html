Node.js

Node.js is a packaged compilation of Google's V8 JavaScript engine, the libuv platform abstraction layer, and a core library, which is itself primarily written in JavaScript. Node.js is differnt then the other trandtion server side language:
  • It's written in C and C++ which make it really fast.
  • its highly scalable.
  • one script handles all the communication with the respective clients.
  • High performance.
  • Asynchronous event driven server-side JavaScript.
use non-blocking, event-driven I/O to remain lightweight and efficient in the face of data-intensive real-time applications that run across distributed devices.
